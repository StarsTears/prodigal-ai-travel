# 三山五岳旅游知识库 · 总览与目的地画像

## 范围说明（供检索统一口径）

旅游语境下常说的“三山五岳”，通常按以下口径处理：  
- **三山**：黄山、庐山、雁荡山  
- **五岳**：东岳泰山、西岳华山、南岳衡山、北岳恒山、中岳嵩山

当用户仅提“三山五岳”，助手应先按该口径回答；若用户有特定版本（如神话三山），再按用户口径重排。

## 八大山岳一句话画像

- **黄山（安徽黄山）**：奇松怪石云海温泉，摄影与徒步兼顾，索道体系成熟。
- **庐山（江西九江）**：避暑山城气质明显，山上公路与景区点位分散，适合慢游。
- **雁荡山（浙江温州）**：灵峰灵岩大龙湫为核心，峰林与瀑布组合，适合短假。
- **泰山（山东泰安）**：文化权重高，夜爬看日出是经典体验，台阶连续上升强度较高。
- **华山（陕西渭南）**：险峻著称，长空栈道等项目关注开放与安全规则。
- **衡山（湖南衡阳）**：南岳大庙+祝融峰组合，人文与登山双线可并行。
- **恒山（山西大同）**：与悬空寺联游价值高，北方气候季节波动明显。
- **嵩山（河南登封）**：少林寺文化认知度高，适合亲子与人文线路结合。

## 旅行决策主维度

给用户推荐目的地时，优先按以下维度筛选：
1. **体力等级**：轻松（缆车+短步行）/中等（半天到全天徒步）/高强度（长台阶或长距离）
2. **偏好类型**：自然风光 / 历史文化 / 宗教祈福 / 摄影打卡
3. **时间长度**：周末 2-3 日 / 小长假 4-5 日 / 深度 6-8 日
4. **同行人群**：亲子、老人、摄影爱好者、户外进阶用户

## 组合线路建议（高频）

- **华东山岳线（4-6 日）**：黄山 + 庐山（或黄山 + 雁荡山）
- **中原文化线（3-5 日）**：嵩山 + 泰山
- **北方人文线（3-4 日）**：恒山 + 悬空寺 + 大同古城
- **华中祈福线（3-4 日）**：衡山 + 岳阳/长沙城市段
- **挑战型短线（2-3 日）**：华山单点深度（含夜间或清晨时段）

## 给助手的检索提示

当用户问“第一次去三山五岳怎么选”时，先追问：  
**出发城市、可用天数、体力水平、是否带老人小孩、最想看日出/云海/人文哪一类**。  
再从八大山岳中筛选 1-2 个同区域目的地，避免跨区折返导致交通时间挤压游玩时间。
