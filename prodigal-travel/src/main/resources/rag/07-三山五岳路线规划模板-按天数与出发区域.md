# 三山五岳旅游知识库 · 路线规划模板（按天数与出发区域）

## 规划原则

1. 先定 **可用天数**，再定 **区域内组合**，最后补景点细节。  
2. 单次行程优先同一大区（华东/华中/华北/西北），避免跨区拉车。  
3. 长途抵达首日只做轻量活动，重体力山岳放在 D2 或 D3。

## 按天数模板

### 2-3 日（周末）

- **模板 A（单山深度）**：D1 抵达 + 轻游；D2 核心登山；D3 返程
- **模板 B（人文+轻徒步）**：D1 城市文化段；D2 山岳景区；D3 半日补点返程
- **适配目的地**：华山、泰山、雁荡山、衡山

### 4-5 日（小长假）

- **模板 C（双目的地）**：D1 抵达；D2 目的地 1；D3 交通切换；D4 目的地 2；D5 返程
- **适配组合**：黄山+庐山、嵩山+泰山、衡山+长沙城市段
- **节奏建议**：D3 作为机动日，防天气或排队带来的连锁延误

### 6-8 日（深度假期）

- **模板 D（区域环线）**：2-3 个山岳 + 1-2 个城市缓冲段
- **适配组合**：华东三山环线（黄山-庐山-雁荡山）或五岳分段打卡
- **节奏建议**：每 2 天至少安排 0.5 天低强度活动

## 按出发区域推荐

### 华东出发（上海/杭州/南京/合肥）
- 优先：黄山、雁荡山、庐山
- 交通策略：高铁优先，减少机场进出与二次转运

### 华北出发（北京/天津/石家庄）
- 优先：泰山、嵩山、恒山
- 交通策略：高铁+短途接驳，恒山常与大同联游

### 华中出发（武汉/长沙/郑州）
- 优先：衡山、嵩山、庐山
- 交通策略：中短途高铁为主，可做双山组合

### 西北出发（西安及周边）
- 优先：华山单点深度或华山+洛阳/郑州方向
- 交通策略：先做西北单点，再考虑跨区二段式行程

## 常见失败规划（给助手规避）

- 一次行程塞入 3 座以上非同区域山岳，导致每天在路上。
- 连续两天高强度登山，第三天被迫大幅压缩景点。
- 回程当天还安排远离机场/高铁站的重景点，误车风险高。

## 给助手的检索提示

输出路线时使用固定格式：  
**每日主目的地 + 上午/下午安排 + 交通方式占位 + 体力等级 + 住宿建议区域**。  
若用户未给体力信息，默认中等强度并提供“轻松版/挑战版”双方案。
