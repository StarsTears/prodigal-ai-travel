# 三山五岳旅游知识库 · 逐山详解与游玩强度

## 三山

### 黄山
- **核心看点**：迎客松、光明顶、始信峰、西海大峡谷
- **游玩建议**：2 日更从容；1 日需高效使用索道并控制步行段
- **体力等级**：中-高（节假日排队会放大体力消耗）
- **注意事项**：雨雾天气多，防滑与保暖都要准备

### 庐山
- **核心看点**：含鄱口、五老峰、花径、锦绣谷、牯岭镇
- **游玩建议**：2-3 日慢游更适合；山上换乘与点位间距离需预留时间
- **体力等级**：轻-中（可通过观光车降低步行强度）
- **注意事项**：旺季住宿需提前订，避暑季人流明显上升

### 雁荡山
- **核心看点**：灵峰夜景、灵岩飞渡、大龙湫瀑布
- **游玩建议**：2 日经典；可与温州城市段拼接成周末线
- **体力等级**：中
- **注意事项**：雨天瀑布景观更好，但山路湿滑风险增加

## 五岳

### 泰山（东岳）
- **核心看点**：红门到南天门古登山线、玉皇顶、日出
- **游玩建议**：夜爬+日出是高频玩法，次日建议减量安排
- **体力等级**：中-高（台阶连续）
- **注意事项**：夜间温差大，头灯与保暖层必备

### 华山（西岳）
- **核心看点**：东峰日出、西峰险峻山势、长空栈道（以景区开放为准）
- **游玩建议**：1-2 日；索道上山可显著降低体力门槛
- **体力等级**：高（不坐索道时）
- **注意事项**：高风与雨雪天气下，部分线路可能临时管制

### 衡山（南岳）
- **核心看点**：南岳大庙、祝融峰、藏经殿
- **游玩建议**：2 日可覆盖人文+登峰；宗教活动日人流更集中
- **体力等级**：中
- **注意事项**：香期与节庆时段关注交通组织与客流管控

### 恒山（北岳）
- **核心看点**：恒山主景区、悬空寺联游
- **游玩建议**：1-2 日；常与大同古城、云冈石窟组合
- **体力等级**：中
- **注意事项**：冬季及大风天气体感温度低，需防风装备

### 嵩山（中岳）
- **核心看点**：少林寺、塔林、三皇寨或嵩阳景区
- **游玩建议**：2 日舒适；亲子游客可提高人文比重、降低徒步强度
- **体力等级**：轻-中
- **注意事项**：演出、讲解、研学团时段可能影响排队时长

## 强度分级（给行程规划器）

- **轻松型**：庐山、嵩山（以人文为主时）
- **均衡型**：雁荡山、衡山、恒山
- **挑战型**：黄山、泰山、华山

规划多山连线时，不建议连续两天都安排挑战型山岳；至少插入一个交通休整或城市慢游半天。
