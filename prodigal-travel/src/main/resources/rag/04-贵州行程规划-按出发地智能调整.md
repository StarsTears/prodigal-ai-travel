# 贵州旅游知识库 · 行程规划（按用户所在地调整）

## 核心原则：先问清「从哪来、到哪去」

1. **用户所在地（或出发城市）** 决定 **大交通方式**（航空/高铁/自驾）与 **疲劳程度**：跨省长途抵达首日应 **低密度、少换乘、早休息**。  
2. **用户在贵州省内何处** 决定 **省内枢纽**：以 **贵阳北/贵阳东、遵义、凯里南、安顺西、都匀东、铜仁** 等为节点做动车辐射，避免单日车程超过 **4～5 小时**（不含游玩）。  
3. **必去景点** 决定 **入境机场/火车站优先级**：不要一律「先飞贵阳」，可按焦点目的地 **就近入境** 减少折返（见下表）。

## 按出发地类型的调整逻辑（须调用工具辅助）

- **调用 `getSmartGuizhouPlanGuidance`（或同类工具）**：根据用户出发城市与贵阳的 **大致直线距离**，自动给出 **高铁优先 / 航空优先 / 省内自驾** 的倾向，并给出 **首日轻松、末日上午购物或短游** 的建议。  
- **模型职责**：将工具返回的 **交通倾向** 与上表 **枢纽**、知识库中的 **景点强度** 合成 **按日表格**，明确每天 **主活动区域 + 交通方式占位**（无地图 MCP 时不写精确公里数，可写「约半日车程」等定性描述）。

## 行程节奏模板（需按出发地压缩或拉伸）

### 省外经典 5 日骨架（贵阳集散示例）

- **D0/D1（抵达日）**：住贵阳，甲秀楼/黔灵山浅游或休息；**不安排**梵净山长途拉车。  
- **D2**：贵阳—黄果树一线，住安顺或返回贵阳（视次日动线）。  
- **D3**：动车/包车向荔波或凯里方向（二选一，勿贪多）。  
- **D4**：深度游玩选定片区（小七孔全日 **或** 西江+夜景）。  
- **D5**：返程交通日，**末段只安排**机场/车站 1 小时圈内活动。

### 黔东南偏重 4 日（适合珠三角/长三角飞贵阳后）

- **D1**：抵达贵阳，动车至 **凯里南**，短驳西江，夜景。  
- **D2**：西江—镇远（汽车/包车为主），古城夜游。  
- **D3**：镇远—贵阳或继续延伸（视返程）。  
- **D4**：返程。

### 铜仁入境梵净山优先 3 日

- **D1**：抵达 **铜仁**，休整、适应海拔。  
- **D2**：梵净山全日（备雨具与预约）。  
- **D3**：返程或衔接凤凰（省外用户常跨省联游，须在合规前提下单独说明）。

## 与 MCP 地图的配合

若已启用 **地图 MCP**：应用其计算 **出发地—枢纽—景区** 的时长，替换本节定性描述中的「约」「量级」等词，并标注 **收费站/限行** 风险提示（以地图服务为准）。

## 给助手的检索提示

用户未说明出发地时，**必须先追问**「您从哪个城市出发、计划几天、必去哪些地方」；再 **调用基于出发地的规划工具**，最后结合 RAG 景点段落生成 **可执行的逐日行程**，并在首尾日体现 **智能减负**（长途抵达后不排高强度登山）。
